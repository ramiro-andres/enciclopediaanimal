# Enciclopedia Animal

Atlas veterinario con razas, nutrición, enfermedades y diccionario médico.

## Sitio publicado

https://ramiro-andres.github.io/enciclopediaanimal/

## Desarrollo local

```bash
bash iniciar.sh
# Abrir http://localhost:8080
```

## Regenerar datos

```bash
bash actualizar_datos.sh
```

## Pruebas

```bash
bash ejecutar_pruebas.sh
```

## Despliegue

El workflow `.github/workflows/deploy-pages.yml` ejecuta pruebas y publica en GitHub Pages.

**Requisito en GitHub:** Settings → Pages → Build and deployment → **GitHub Actions**.
