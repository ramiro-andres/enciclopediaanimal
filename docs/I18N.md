# Internacionalización (i18n) — Atlas Animal

**Historia:** US-UX-07  
**Fase 1 (este sprint):** UI estática en ES/EN  
**Fase 2 (futuro):** Traducción de 400+ fichas clínicas

## Estrategia

| Capa | Enfoque |
|------|---------|
| Cromo UI (botones, etiquetas, navegación) | Claves en `js/i18n.js` (`I18n.t('key')`) |
| Contenido clínico (`enciclopedia.json`) | Español único por ahora — volumen y revisión veterinaria |
| Glosario médico | Español — términos técnicos con equivalencias EN en fase 2 |

## Uso

```javascript
I18n.setLang('en');  // persiste en localStorage (atlas_lang)
I18n.t('nav.home');  // "Home"
```

Elementos HTML estáticos usan `data-i18n="clave"` y se actualizan con `I18n.apply()`.

## Decisión para fichas clínicas

**JSON paralelo** (`enciclopedia.en.json`) descartado en fase 1 por duplicar ~2 MB y 2 500+ enfermedades.  
**Recomendación fase 2:** campos opcionales `_en` por raza o pipeline de traducción asistida con revisión humana obligatoria.
